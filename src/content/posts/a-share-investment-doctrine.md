> **一句话总纲：在 A 股，你赚的是"对周期判断对"的钱，不是"陪伴伟大公司"的钱。下面所有规则，都是这句话的展开。**

## 一、地形认知（一切的前提）

A 股 / 中国市场结构性地缺少"定价权 + 稀缺"的护城河型 compounder——没有海力士、没有英伟达式的永续复利。原因有三：

1. **内卷**——催生行业的机制，和把毛利卷没的机制，是同一个。
2. **稀缺环节不可得**——真正像海力士的资产（长鑫、长存）没上市；最先进的算力被制裁卡住。
3. **中国的强项在浅层**——你能买到的，恰恰是 IP 门槛最低、靠执行和堆量、最容易内卷的那一层。

**核心推论：护城河深度 与 "A 股能买到" 反相关。** 深的被地缘和上市结构挡在外面，留给你的天然是浅的。

→ 不要在 A 股找"可以长拿陪伴的伟大公司"。你的身份是：**周期交易者 + 稀缺资产收租人。**

## 二、能不能"持有"——护城河三问

任何想长拿的标的，先过三关，缺一不可。

### 1. 稀缺 / 不可复制？

护城河背后是"建不出第二个"的东西（稀缺资源、管制垄断位置），还是只是"更便宜 / 更高效 / 先发"？后者会被下一个中国玩家追平。

- 浅：中际旭创＝执行＋先发，且有 CPO 终值悬顶。
- 深：格林布什 / 长电＝不可复制。

### 2. 定价权 还是 只是生存？

它能定价（寡头、稀缺），还是只是低成本 producer——能熬死对手、但盈利仍由周期价格决定？后者不是 compounder，是"择时择得好的周期股"。

- 海力士＝定价权；天齐＝生存护城河（2024 巨亏 79 亿、2025 才扭亏 4.6 亿，护城河保命，但挡不住周期）。

### 3. 报表能不能对你撒谎（可造假性）？

营收＝真实终端消费（物理计量、不可掩盖，如长电的发电量、国网应收），还是走一条能压货的渠道（经销商，如白酒）？

- 可掩盖的生意会"利润表年年增、真实动销三年前已烂"，等爆出来就晚了。
- **真相不在利润表，在批价倒挂、渠道库存、合同负债里。**

→ 三关全过才可长拿。A 股几乎无一全过；长电式收租资产是稀有例外，且它也只是"类债券＋小幅成长"，不是 compounder，还必须便宜时买。

## 三、黑暗＝买，还是黑暗＝走——周期 vs 长期衰退

"看不到希望"是个陷阱信号：它既是周期底部的买点，也是长期衰退的卖点。唯一的判别问题：

> **有没有回归机制？**

- **周期（有机制）**：成本曲线托底 / 资本开支停止 → 供给出清 / 需求刚性。机制保证它会翻过来——你不需要看见复苏。→ **黑暗＝买**（锂）。
- **长期衰退（无机制）**：需求基本盘本身在不可逆收缩（人口、文化、制度）。没有出清能把人请回来。→ **黑暗＝走，坚持没有意义**（白酒：Z 世代不接、反腐永久压制商务场景、社交硬通货瓦解）。

"别人恐惧我贪婪"只有在回归机制（前提）存在时才成立；否则只是没有底的下落。

长期衰退里的情绪反弹＝**带止损的交易**，不是持仓。别把它错认成核心资产。

## 四、恶性竞争 / 价格战

**战时：一个都不留。** 所有人毛利被打穿、胜负未知，你只是在给篝火添柴（外卖战：三家一年烧 1730 亿＝行业巅峰五年利润，无赢家；连守住第一的也赔进一年利润）。

**再评估点在仗的尾巴**：收尾成"理性寡头"（活下来那个重获定价权 → 可在情绪低点考虑）还是"多巨头永久对峙"（谁都不退 → 长期回避）？

多个不差钱的巨头抢同一个入口、谁都不退 → 接近永久内卷 → 即便"停战"反弹也是交易，不是"护城河修复、可长拿"。

反讽提醒：美团自己是靠打赢千团大战成的。所以规则是 **"不在打的时候碰"，不是"价格战行业永不可碰"。**

## 五、预警系统（信号纪律）

凡是利润表会撒谎或滞后的，就在**不可掩盖的领先信号**上建预警，抢在"确认"之前动：

- **锂**：海外评估价（PRA）、上游拍卖、producer tone。
- **白酒**：批价 / 价格倒挂、渠道库存、合同负债（预收款）。
- **共性**：盯真实需求在打到营收之前露头的地方。

### 设计原则

- **窗口化＋衰减**：滚动窗口里异步累加，不要求同周共振。
- **按质量 / 独立性加权**：不等票；警惕同源信息被数三遍。
- **带幅度**：不要纯二值（温和阴跌 ≠ 折价扩大 / 流拍）。
- **阈值按动作的代价 / 可逆性分级**：便宜且可逆（暂停加仓）＝一个信号就动；昂贵且难逆（砍核心仓 / 转红）＝要独立信号 ＋ 基本面或公司硬触发。
- **对称的复位规则**：定义清楚什么情况降级，不只定义升级。
- 用**成本曲线**锚定"下方还有多少空间"。
- 记录命中率，用历史回放调阈值，不拍脑袋。

→ 系统的用途不是守护信仰，是在该走的时候走。**及时离场，才是 A 股真正给钱的本事。**

## 六、操作模型（组合层面）

你只做四件事：

1. **周期底部交易**：有机制的周期，成本曲线底部进，预警器择时退（锂、天齐）。
2. **收稀缺资产租**：不可造假的稀缺资产，便宜时买，当"类债券＋小幅成长"（长电）。
3. **混合体期权**：成本底盘白捡的创新期权——但必须分清自己在 underwrite 哪个引擎，两者风险完全不同（联邦：API / 胰岛素成本盘 ＋ GLP-1 创新盘）。
4. **其余一切＝信号一翻就走的交易**：不为希望持有，不在战时持有。

### 纪律护栏

- 约 1/3 现金。
- 一年 1–2 笔高确信。
- 多源交叉验证。
- 分批 / 部分止盈。
- 警惕风格漂移（买便宜但无催化＝同道猎聘式错误 / 白酒陷阱）。
